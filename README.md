# insta_clone_laravel
Instagram clone using Laravel Framework

# Application Name
> freeGram

# Description
This is an Instagram clone using Laravel Framework.
Its fully working clone with the following functionality : 
  1. Login/SignUp
  2. Create Post
  3. Update Profile
  4. Follow/Unfollow
  5. Add Comments
  6. Welcome mail
  and etc.
  
 It uses SQLite databasse for backend.
